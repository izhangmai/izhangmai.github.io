# git常用功能介绍

## 1 搭建测试环境
开源中国(https://git.oschina.net/)提供了一个免费的git server服务器，可以到那里注册账号并托管自己的项目，同时也是学习git很理想的一个环境。
## 2 代码下载/同步/提交
下载：git clone https://git.oschina.net/brc/LfListView.git
同步：git pull
提交：git add --all;   git commit -m "xxx";  git push;
